# ModbusRTUSlave Example Sketch
This example demonstrates how to setup and use the ModbusRTUSlave library.

## Hardware Required
- 2 Arduino Boards
- 4 Pushbutton Switches
- 8 LEDs
- 8 330 Ohm Resistors
- 4 10K Ohm Potentiometers
- Wires
